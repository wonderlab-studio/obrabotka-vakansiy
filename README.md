# Обработка вакансий — сервис-помощник в поиске работы

Веб-сервис, который помогает соискателю быстрее и качественнее откликаться на вакансии hh.ru:

1. По ссылке на вакансию извлекает требования, информацию о компании, задачи и бонусы.
2. Сопоставляет тезисы опыта пользователя с требованиями вакансии, выделяя совпадения и пробелы.
3. Генерирует готовое сопроводительное письмо на основе этого сопоставления.

Интерфейс — одностраничный веб-сервис без регистрации: вставил ссылку → увидел разбор
вакансии и сопоставление опыта → получил сопроводительное письмо для копирования.

## Стек

- **Бэкенд**: Node.js + Express (`server/`) — парсинг страницы вакансии hh.ru (`hhParser.js`)
  и вызовы LLM (`gemini.js`).
- **Фронтенд**: статический HTML/CSS/JS (`public/index.html`), без фреймворка.
- **LLM**: Google Gemini API с фолбэк-цепочкой `gemini-2.5-flash` → `gemini-2.5-flash-lite` →
  облачная Ollama → локальная Ollama.

## Запуск в облаке

Требует заполнения переменных: GEMINI_API_KEY, GEMINI_MODEL, GEMINI_MODEL_LITE, OLLAMA_CLOUD_API_KEY, OLLAMA_CLOUD_MODEL.

## Запуск локально

```bash
npm install
cp .env.example .env   # заполнить GEMINI_API_KEY и остальные переменные
npm start               # http://localhost:3000
```

## Переменные окружения для локального запуска

См. `.env.example`. Обязательна как минимум `GEMINI_API_KEY`; остальные — опциональные звенья
фолбэк-цепочки LLM.
